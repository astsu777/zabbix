# Introduction
This repository contains custom templates for *Zabbix*. It also contains configuration files for the agents, servers and custom scripts for enhanced monitoring.


# Structure

Here is the file structure you can find:

- **agent**: this directory contains custom configuration files for the agent
- **server**: this directory contains custom configuration files for the server
- **templates**: this directory contains all the custom templates

# Resources

Here are some useful resources you might want to check:

- [Zabbix Documentation](https://www.zabbix.com/documentation/current/en/manual): general *Zabbix* documentation where all the necessary information are available
- [Zabbix Templates Import Tutorial](https://www.zabbix.com/documentation/current/en/manual/xml_export_import/templates#importing): this part of the documentation explains how to import templates
